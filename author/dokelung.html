Don't Reinvent Sandwich - Python 的裝飾器與環境管理器

日 25 6月 2017

今年的 Pycon Taiwan，有一篇我滿感興趣的演講：Don't Reinvent Sandwich - Python Decorator and Context Manager，講者是蕭聖穎先生。 該演講的 投影片 和講者的 Facebook。 由於我對於語言本身的設計和使用慣例非常感興趣，所以就去朝聖了一下。講題的主旨大略是：我們通常透過抽取一段重複出現的 連續代碼 成為 函數 (function) 或 類別 (class) 來重複利用他，避免重複發明輪子。但有的時候想要抽取的部分並非 中間 ( 三明治的麵包）連續的部分，而是 頭尾 （三明治的麵包），這個時候我們便需要使用一些其他的手法或機制來達到 reuse 的目的了。Python 正好非常友善地提供了 裝飾器 (decorator) 和 環境管理器 (context manager)，可以很容易地達成我們的目的 ...
